# AI Coding Agent

A simple autonomous Python agent that:
- Reads a coding task
- Generates Python code using OpenAI
- Executes the code
- Any errors are researched on the Web
- Errors are corrected and tasks are reattempted automatically

With:
- Python agent runtime
- LLM prompt → code generation loop
- Retry logic
- Subprocess execution sandbox
- Logging system
- Installable Python package
- CLI tool
- Standalone EXE distribution
- Runtime environment detection
- Interpreter fallback logic
- File system-safe execution
- End-to-end working demo

## Prerequisites 
- An OpenAI API key. You may edit the enclosed .env file or be prompted for it at runtime.
- A task for the AI agent to perform in task.txt. A sample task is included.
